API Reference

Types

S2Geography.S2Geog — Type
S2Geog{Trait}

A wrapper around an S2GeogGeography pointer. Parameterized by GeoInterface geometry trait for type-stable dispatch.

Automatically frees the underlying C object via a finalizer.

Measurements

S2Geography.distance — Function
distance(geom1, geom2) -> Float64

Compute the spherical distance (in radians) between two geometries. Multiply by Earth's radius (~6371 km) to get distance in km.

source
S2Geography.area — Function
area(geom::S2Geog) -> Float64

Compute the area of a geography in steradians. Multiply by Earth's radius squared (~6371² km²) to get area in km².

source
